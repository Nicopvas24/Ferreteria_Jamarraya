# Imágenes - Página Quiénes Somos

Las imágenes de la página "Quiénes Somos" se cargan desde esta carpeta de forma fija. Para cambiar las imágenes, reemplaza los archivos correspondientes:

## Archivos esperados

- **tienda.jpg** - Foto de la tienda/local (sección Historia)
- **juan.jpg** - Foto de Juan Pérez (Fundador)
- **maria.jpg** - Foto de María López (Directora Comercial)
- **carlos.jpg** - Foto de Carlos Restrepo (Jefe de Bodega)
- **galeria-1.jpg** - Primera foto de la galería
- **galeria-2.jpg** - Segunda foto de la galería
- **galeria-3.jpg** - Tercera foto de la galería
- **galeria-4.jpg** - Cuarta foto de la galería

## Instrucciones

1. Reemplaza cualquiera de estos archivos con tus propias imágenes
2. Mantén el mismo nombre de archivo
3. Las imágenes se actualizarán automáticamente en la página
4. Los cambios se realizan **únicamente desde el código/carpeta**, no desde la página web

## Formatos soportados

- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## Notas

- Las imágenes del equipo se muestran en la sección "Equipo"
- Las imágenes de la galería se muestran en la sección "Galería"
- No es posible cambiar las imágenes desde la interfaz web por razones de seguridad y control
