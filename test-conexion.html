4️⃣ Cómo Revisar en Developer Tools

1. Abre DevTools: F12
2. Ve a la pestaña "Network"
3. Intenta Iniciar Sesión o Registrarte
4. Busca la request a "usuarios.php"
5. Haz clic y revisa:
   - Status: debe ser 200
   - Response: debe ser JSON válido
   - Preview: muestra la respuesta formateada
